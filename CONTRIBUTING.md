# Contributing

Contributions are welcome!

If you have an idea that makes this project better, feel free to contribute it. You have complete freedom to improve the design, code, animations, performance, responsiveness, accessibility, documentation, or anything else.

## How to Contribute

1. Fork the repository.
2. Make your changes.
3. Test your changes.
4. Open a Pull Request explaining what you changed.

That's it.

## New 404 Pages

If your idea is a completely new 404 page with a different design or concept, please [open an issue](../../issues) first. It may be better suited as a new repository in the [Not Found Pages](https://github.com/notfoundpages) organization.
